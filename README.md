# KYT-Mock-API

Mock API for `UserApp` and `FrankieOne ` services

**Requirement**:\
`NodeJS` install via https://nodejs.org/en/download/ \
**OR**\
`$brew install node`

## Usage
Run `$node app.js` in the each service folder you want to run the mock API
 
### Note
Default `PORT`:
  - UserApp - `3031`
  - FrankieOne - `3032`
